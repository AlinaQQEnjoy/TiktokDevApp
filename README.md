# Short-Form Video Manager Pages

This repository contains three static pages that can be published with GitHub Pages:

- `index.html`: official website URL
- `terms-of-service.html`: Terms of Service URL
- `privacy-policy.html`: Privacy Policy URL
- `callback.html`: OAuth callback URL

After GitHub Pages is enabled, use these URLs in the TikTok developer form:

- Website: `https://alinaqqenjoy.github.io/TiktokDevApp/`
- Terms of Service: `https://alinaqqenjoy.github.io/TiktokDevApp/terms-of-service.html`
- Privacy Policy: `https://alinaqqenjoy.github.io/TiktokDevApp/privacy-policy.html`
- Callback URL: `https://alinaqqenjoy.github.io/TiktokDevApp/callback.html`

The contact email used on these pages is `qtangalina@gmail.com`.
